Drink Tracking App

GitHub
www.drinktracker.net (Currently beta testing: send me an email for access)

This is my first mobile app, built with Next.js, using Vercel for deployment/hosting and Supabase as the backend database. It helps users track, visualize, and share their alcohol consumption habits. The community sharing feature aims to help users support each other in being accountable for their goals. I had no prior React or JavaScript experience and built most of it with assistance from Claude. I'm amazed at how quickly a single person with little experience can build things with LLMs.
